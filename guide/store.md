全局状态文件存放在 `/src/store/` 目录下，请按模块进行区分。同时请保证文件名和文件内唯一ID保持一致，建议使用 `pnpm new` 指令进行创建。

例如新建一个 `example.ts` 的文件：

```ts
export const useExampleStore = defineStore(
  // 唯一ID
  'example',
  {
    state: () => ({}),
    getters: {},
    actions: {},
  },
)
```

使用方法：

```ts
const exampleStore = useExampleStore()

exampleStore.data
exampleStore.doSomething()
```
