# Fantastic-template

## 这是什么

一款开箱即用的 Vue 项目模版，基于 Vue(3.x) & Vite(2.x)

## 特点

- 默认集成 vue-router 和 vuex
- 全局 SCSS 资源引入
- 全局组件自动注册
- 支持 SVG 图标，CSS 精灵图自动合成
- 支持 mock 数据，可摆脱后端束缚独立开发
- 支持 gzip / brotli 优化项目体积，提高加载速度
- 结合 IDE 插件、ESlint 、stylelint 、Git 钩子，轻松实现团队代码规范
